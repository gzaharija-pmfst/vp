Vježba 08

  • Cilj vježbe je upoznati se sa još jednom vrstom grafa
    • box-plot ili whisker-plot (kutijasti dijagram)
      • predstavlja 5 karakteristika
        • minimum i maksimum (bez outlier-a)
        • medijan
        • prvi i treći kvartil

Učitavanje skupa podataka

  • Prvi korak je učitavanje podataka
    • koristimo već poznati skup podataka
    • morati ćemo ga modificirati kako bi odgovarao našem prikazu
      • 365 dana ćemo rasporediti u 12 mjeseci
      • moramo parsirati datum i napraviti metodu za dohvat broja mjeseca
  const dataset = await d3.json("/../vrijeme.json")

  const datumParser = d3.timeParse("%Y-%m-%d")
  const brojMjeseca = d3.timeFormat("%m")

Učitavanje skupa podataka

  • Podatke ćemo grupirati koristeći d3.groups() metodu koja po odabranom ključu grupira skup podataka
    • metoda prima dva parametra
      • skup koji grupiramo
      • ključ po kojem ćemo ga grupirati
  const podaciPoMjesecu = d3.groups(dataset, d => brojMjeseca(datumParser(d.date)))
  console.log(podaciPoMjesecu)

Učitavanje skupa podataka

  • Rezultat nam je niz sa 12 elemenata
    • svaki element sadrži dva podataka
      • ključ po kojem smo grupirali
      • pod-niz sa svim pripadajućim podacima

Učitavanje skupa podataka

  • Potrebna su nam još dvije accessor funkcije
    • jedna koja dohvaća željenu metriku koju ćemo prikazati pomoću box plota
      • u ovom slučaju temperatureMax
      • druga će nam biti za dohvaćanje broja mjeseca
  • Proširiti ćemo naš skup podataka sa podacima potrebnim za iscrtavanje box-plota za odabranu metriku
    • možemo unaprijed napisati accessor 
const yAccessor = d => (d.temperatureMax - 32) * 0.5556;
const mjesecAccessor = d => d.mjesec

Učitavanje skupa podataka

  • Podaci koji nam trebaju su
    • prvi i treći kvartil
      • koristiti ćemo metodu d3.quantile()
    • medijan - d3.median()
    • interkvartilna vrijednost (IQR) - razlika q3 i q1
    • minimalne i max vrijednosti
      • sa i bez outliera (koje ćemo posebno izdvojiti)
    • dodati ćemo i brojčanu oznaku mjeseca
  • Sve navedene podatke ćemo dodati u 12 elemenata koji predstavljaju podatke po mjesecima

Učitavanje skupa podataka

  • Konačan rezultat izgleda ovako:
  const poMjesecuProsireno = podaciPoMjesecu.map(mjesec => {
    const mjesecTemp = mjesec[1].map(yAccessor).sort((a,b) => a - b)
    const q1 = d3.quantile(mjesecTemp, 0.25)
    const median = d3.median(mjesecTemp)
    const q3 = d3.quantile(mjesecTemp, 0.75)
    const iqr = q3 - q1
    const [min, max] = d3.extent(mjesecTemp) 
    const rasponMin = d3.max([min, q1 - iqr * 1.5])
    const rasponMax = d3.min([max, q3 + iqr * 1.5])
    const outliers = mjesec[1]
      .filter(d => yAccessor(d) < rasponMin || yAccessor(d) > rasponMax)

    return {
      ...mjesec,
      mjesec: +mjesec[0],
      q1, median, q3, iqr, min, max, rasponMin, rasponMax, outliers
    }
  })

Crtanje grafa

  • Dimenzije grafa i iscrtavanje okvira i granica su standardni...
// 2. Dimenzije grafa
const sirina = 800
let dimenzije = {
  sirina: sirina,
  visina: sirina * 0.6,
  margine: { top: 30, right: 10, bottom: 30, left: 50,},
}
dimenzije.grSirina = dimenzije.sirina 
  - dimenzije.margine.left - dimenzije.margine.right
dimenzije.grVisina = dimenzije.visina 
  - dimenzije.margine.top - dimenzije.margine.bottom

// 3. Crtanje grafa
const okvir = d3.select("#okvir")
  .append("svg")
    .attr("width", dimenzije.sirina)
    .attr("height", dimenzije.visina)

const granice = okvir.append("g")
    .style("transform", 
           `translate(${dimenzije.margine.left}px, ${dimenzije.margine.top}px)`)

Definiranje razmjera

  • Idući korak nam je definiranje razmjera
    • po x osi ćemo imati vrijednosti od 1-13 (granice stupaca za mjesece)
    • domena y osi su minimalne i maksimalne vrijednosti za našu metriku (svojstvo maxTemperature koju pretvaramo u °C)
const xSkala = d3.scaleLinear()
  .domain([1, podaciPoMjesecu.length + 1])
  .rangeRound([0, dimenzije.grSirina])
  .nice()

const ySkala = d3.scaleLinear()
  .domain(d3.extent(dataset, yAccessor))
  .range([dimenzije.grVisina, 0])
  .nice()

Definiranje razmjera

  • Sjetite se izrade histograma
    • također smo grupirali podatke u grupe/košare
    • koristili smo d3.histogram()
      • dobijemo x0 i x1 svojstva koji su granice podataka
    • iskoristiti ćemo d3.bin() (nova verzija histogram metode)
    • podijeliti ćemo podatke u 12 kategorija, s obzirom da su nam mjeseci cijeli brojevi od 1-12 ne bi trebalo biti problema sa tresholds parametrom
const generatorGrupa = d3.bin()
  .value(mjesecAccessor)
  .thresholds(podaciPoMjesecu.length)  // 12
 
const grupe = generatorGrupa(poMjesecuProsireno)

Iscrtavanje podataka

  • Možemo početi iscrtavati podatke
  • Algoritam nam je poznat
    • odaberemo sve željene elemente
      • pridružimo im podatke
        • izbacimo stare elemente
        • stvorimo nove elemente
    • spojimo postojeće i nove elemente
const paddingStupci = 7
let sviStupci = granice.selectAll(".stupac")
  .data(grupe)  

sviStupci.exit()
    .remove()  

const noviStupci = sviStupci.enter().append("g")
    .attr("class", "stupac")
  
sviStupci = noviStupci.merge(sviStupci)

Iscrtavanje podataka

  • Sada imamo sve za crtanje elemenata
    • početi ćemo od linija koje idu od minimalne do maksimalne vrijednosti
      • pokušajte sami svakom stupcu dodati line element
      • linija mora biti na sredini stupca (znamo raspone po x osi)
      • početak i kraj linije su minimalne i maksimalne vrijednosti
const sirinaStupca = bar => xSkala(bar.x1) - xSkala(bar.x0)
const rasponLinije = sviStupci.append("line")
    .attr("class", "linija")
    .attr("x1", d => xSkala(d.x0) + sirinaStupca(d) / 2)
    .attr("x2", d => xSkala(d.x0) + sirinaStupca(d) / 2)
    .attr("y1", d => ySkala(d[0].rasponMin))
    .attr("y2", d => ySkala(d[0].rasponMax))

Iscrtavanje podataka

  • Idući element su pravokutnici (rect elementi)
    • početna točka (x,y) je gornji lijevi rub pravokutnika
      • x koordinata je vrijednost mjeseca + pola paddinga
      • y je vrijednost trećeg kvartila
    • visina i širina su također trivijalni
      • širinu stupca već imamo (umanjiti ćemo padding)
      • visina je razlika 1. i 3. kvartila ("provučena" kroz skalu)
const kutije = sviStupci.append("rect")
    .attr("x", d => xSkala(d.x0) + paddingStupci / 2)
    .attr("y", d => ySkala(d[0].q3))
    .attr("width", d => sirinaStupca(d) - paddingStupci)
    .attr("height", d => ySkala(d[0].q1) - ySkala(d[0].q3))

Iscrtavanje podataka

  • Linije medijana su jednostavne:
    • x1 i x2 su rubovi pravokutnika (pazite na padding)
    • y koordinate su iste u oba slučaja (medijan)
const medijani = sviStupci.append("line")
    .attr("class", "medijan")
    .attr("x1", d => xSkala(d.x0) + paddingStupci / 2)
    .attr("x2", d => xSkala(d.x1) - paddingStupci / 2)
    .attr("y1", d => ySkala(d[0].median))
    .attr("y2", d => ySkala(d[0].median))

Iscrtavanje podataka

  • Iscrtati ćemo i "brkove" - horizontalne linije na kraju linija za minimalne i maksimalne vrijednosti
    • logika je slična liniji medijana
      • samo su y koordinate različite
      • možemo ih malo skratiti
const minRasponi = sviStupci.append("line")
    .attr("class", "linija")
    .attr("x1", d => xSkala(d.x0) + sirinaStupca(d) * 0.35) // malo krace
    .attr("x2", d => xSkala(d.x1) - sirinaStupca(d) * 0.35)
    .attr("y1", d => ySkala(d[0].rasponMin))
    .attr("y2", d => ySkala(d[0].rasponMin))
  
const maxRasponi = sviStupci.append("line")
    .attr("class", "linija")
    .attr("x1", d => xSkala(d.x0) + paddingStupci / 2 + sirinaStupca(d) * 0.3)
    .attr("x2", d => xSkala(d.x1) - paddingStupci / 2 - sirinaStupca(d) * 0.3)
    .attr("y1", d => ySkala(d[0].rasponMax))
    .attr("y2", d => ySkala(d[0].rasponMax))

Iscrtavanje podataka

  • Iako nemamo puno outlier podataka, trebali bi i njih iscrtati
  • Ovaj put je logika malo složenija
    • svakom stupcu ćemo dodati novu grupu
      • translatirati ćemo je po x osi na sredinu stupca
    • odabiremo sve circle elemente
      • pridružujemo outlier podatke
        • postavljamo središte po y  osi i radijus
const outliers = sviStupci.append("g")
    .attr("transform", d => `translate(${xSkala(d.x0) + sirinaStupca(d) / 2}, 0)`)
  .selectAll("circle")
    .data(d => d[0].outliers)
    .enter().append("circle")
      .attr("class", "outlier")
      .attr("cy", d => ySkala(yAccessor(d)))
      .attr("r", 2)

Iscrtavanje pomoćne grafike

  • Boxplot elementi su gotovi, sada ćemo dodati oznake
    • poviše svakog elementa možemo ispisati ime mjesece
      • treba nam parser za dobiti ime iz broja
      • zatim ćemo ga formatirati po želji
    • dodati ćemo text element svakom stupcu
      • x pozicija je sredina pravokutnika
      • y je malo iznad početka
      • koristimo transform svojstvo
const parserMjeseca = d3.timeParse("%m")
const formatMjeseca = d3.timeFormat("%b")
const oznake = sviStupci.append("text")
    .attr("class", "oznaka")
    .attr("transform", d => `translate(${xSkala(d.x0) + sirinaStupca(d) / 2}, -15)`)
    .text(d => formatMjeseca(parserMjeseca(d[0].mjesec)))

Iscrtavanje pomoćne grafike

  • Zadnji korak je dodati i vertikalnu os

const yOsGenerator = d3.axisLeft()
    .scale(ySkala)
    .ticks(4)
    
const yOs = granice.append("g")
    .attr("class", "y-os")
    .call(yOsGenerator)
    
const yOsOznaka = yOs.append("text")
    .attr("class", "y-os-oznaka")
